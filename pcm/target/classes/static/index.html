Aim

To understand the principles of Pulse Code Modulation (PCM) and observe the effects of sampling, quantization, and encoding on analog signals.

This experiment demonstrates how continuous analog signals are converted into discrete digital representations through the PCM process, which is fundamental to modern digital communication systems.

Theory

Pulse Code Modulation (PCM) is a method used to digitally represent analog signals. It is the standard form of digital audio in computers, compact discs, digital telephony, and other digital audio applications.

The PCM process involves three main steps:

  • Sampling: The continuous-time analog signal is sampled at regular intervals.
  • Quantization: Each sampled value is approximated to the nearest value within discrete levels.
  • Encoding: Each quantized value is represented in binary or Gray code.

Key Parameters:

  • Sampling Rate (Fs): Samples per second.
  • Quantization Levels (L): Number of amplitude levels.
  • Bit Depth (n): Bits per sample.
  • Bit Rate: Fs × n.

Procedure

  1. Select signal type: Sine, Cosine, or Random.
  2. Adjust amplitude and frequency.
  3. Set sampling frequency and quantization levels.
  4. Select encoding scheme.
  5. Click the SIMULATE button to generate plots.
  6. Observe Channel 1 (Transmitter) signals by clicking different buttons.
  7. Observe Channel 2 (Receiver) signals.
  8. Zoom in/out for clarity using Plotly controls.
  9. Compare input vs output signals to understand quantization error.
